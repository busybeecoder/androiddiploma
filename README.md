# Дипломный проект Шкикавого Ивана 2022 год Университет ИТМО
## PriceHunter

Нативное мобильное приложение для анализа цен цифровых копий компьютерных игр из различных онлайн-магазинов 

## Технологии
1. Kotlin
2. Retrofit
3. Jetpack Compose
4. Hilt
